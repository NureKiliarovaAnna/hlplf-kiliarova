# Пояснення до лабораторної роботи №2

## Виконані завдання

- Рівень 1, завдання 2: створено просту систему управління задачами на Node.js з використанням Express. Реалізовано фронтенд на React для відображення списку задач.
- Рівень 2, завдання 2: додано можливість додавання, редагування та видалення задач. Реалізовано перегляд деталей кожної задачі.
- Рівень 3, завдання 2: реалізовано систему користувачів та ролей доступу. Додано планування задач і нагадування.
- Рівень 4, завдання 2: інтегровано систему електронної пошти для надсилання нагадувань. Реалізовано аналітику виконання задач та звітність.

## Як запустити проєкт

Відкрити два термінали.

### 1. Запуск сервера

cd server
node index.js

### 2. Запуск клієнта

cd client
npm run dev
