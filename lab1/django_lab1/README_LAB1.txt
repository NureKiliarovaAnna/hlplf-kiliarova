Лабораторна робота 1
Python та Django

Виконані завдання для варіанту 10
---------------------------------

Рівень 1, завдання 1:
Створено Django-проект та додаток articles. Додано модель "Стаття" з полями:
- Заголовок
- Текст
- Дата публікації

Рівень 2, завдання 1:
Модель "Стаття" розширено полем "Автор".
Реалізовано сторінку для перегляду всіх статей конкретного автора.

Рівень 3, завдання 1:
Створено шаблони Django для відображення:
- списку статей
- окремої статті
- статей конкретного автора

Рівень 4, завдання 1:
Додано API для отримання статей у форматі JSON.
Для API використовується Django REST Framework.


Запуск локального серверу Django: python manage.py runserver

Основні посилання
-----------------

Головна сторінка зі списком статей:
http://127.0.0.1:8000/

Адмін-панель Django:
http://127.0.0.1:8000/admin/

API зі списком статей у форматі JSON:
http://127.0.0.1:8000/api/articles/

Сторінка окремої статті:
http://127.0.0.1:8000/article/1/

Сторінка статей конкретного автора:
http://127.0.0.1:8000/author/1/
